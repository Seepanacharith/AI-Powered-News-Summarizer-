# 🚀 Features

- 🧠 **AI-Powered Summarization**  
  Generates short and meaningful summaries using NLP and Transformer models.

- 🌐 **Live News Scraping**  
  Fetches real-time headlines from trusted news sources using Puppeteer.

- 🗂️ **Smart Categorization**  
  Automatically classifies articles into categories like Technology, Politics, Sports, Entertainment, and more.

- 🔊 **Voice News Support**  
  Converts summaries into speech using Text-to-Speech APIs for hands-free listening.

- 🎨 **Modern UI/UX**  
  Responsive and animated interface built with React, Tailwind CSS, and AOS animations.

---

# 🏗️ System Architecture

- **News Scraper** → Collects latest news articles from online sources  
- **AI Summarizer** → Generates concise summaries using NLP models  
- **Categorization Engine** → Organizes news into categories  
- **Voice Module** → Converts text summaries into speech  
- **Frontend Interface** → Displays articles, summaries, and interactions  
- **Backend API** → Handles scraping, processing, and API communication

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/Seepanacharith/AI-Powered-News-Summarizer.git
cd AI-Powered-News-Summarizer
```

---

# 💻 Frontend Setup

```bash
cd client
npm install
```

### Install Required Packages

```bash
npm install axios
npm install react-router-dom
npm install @mui/material @emotion/react @emotion/styled
npm install react-icons
npm install framer-motion
```

---

# 🖥️ Backend Setup

```bash
cd server
npm install
```

### Install Backend Packages

```bash
npm install express
npm install axios
npm install cors
```

---

# 🐍 Python Dependencies

```bash
pip install requests
pip install beautifulsoup4
pip install transformers
```

---

# ▶️ Run the Project

## Start Frontend

```bash
npm run dev
```

## Start Backend

```bash
npm start
```
